# 🚀 GeminiPro – AI Voice Assistant

GeminiPro is a full-stack AI-powered voice assistant application built using the MERN stack. It provides an interactive conversational experience with voice recognition, AI-generated responses, user authentication, and personalized assistant settings.

## ✨ Features

* 🔐 User Authentication (Signup/Login)
* 🤖 AI-powered conversations
* 🎤 Voice Recognition
* 🔊 Text-to-Speech responses
* 👤 Personalized AI assistant configuration
* 🖼️ Profile and assistant image upload
* 🔒 Protected routes using JWT authentication
* 📱 Responsive user interface
* ☁️ Cloud deployment support

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/vedchaudhary2005/Gemini.git
cd Gemini
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
FRONTEND_URL=your_frontend_url
GEMINI_API_KEY=your_gemini_api_key
```

---

## ▶️ Run the Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm run dev
```
